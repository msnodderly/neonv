## NeoNV v0.2.0

Quality-of-life improvements focusing on the note list, Org mode rendering, and keyboard workflow.

### New Features

- **Date column in horizontal layout** — See modification dates at a glance with compact row display
- **ISO 8601 date insertion** — Press ⌘D to insert today's date (2026-01-31 format)
- **Org `#+TITLE:` rendering** — Org files now display their title as an H1 in preview

### Fixes

- Fixed focus jumping away when toggling preview mode (⌘P)
- Fixed horizontal layout column order and date truncation
- Removed distracting search term highlighting from the editor

### Internal

- Code quality improvements (SwiftLint cleanup, reduced cyclomatic complexity)

### Installation

Download the DMG, drag NeoNV to Applications. On first launch, right-click → Open (app is unsigned).
