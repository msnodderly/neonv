## NeoNV v1.0.2

Stability and performance improvements for note management and search.

### Fixes

- **Stable note identity** — Cached metadata now uses stable file identifiers, preventing stale data after renames or moves.

### Performance

- **Faster search filtering** — Optimized the search filtering hot path for snappier results with large note collections.

### Installation

Download the DMG and drag NeoNV to Applications, or:

```bash
brew install --cask msnodderly/tap/neonv
```

**First launch (unsigned app):**
1. Double-click NeoNV — macOS blocks with "cannot be opened"
2. Open **System Settings → Privacy & Security**
3. Click **"Open Anyway"** in Security section
4. Enter password when prompted
