## NeoNV v1.0.0

The 1.0 release — NeoNV is feature-complete. A fast, frictionless text capture tool for macOS, inspired by Notational Velocity.

### Core

- **Instant Capture** — One text box, always ready. Auto-saves continuously.
- **Instant Search** — Fuzzy full-text search as you type. Results appear immediately.
- **Zero Friction Storage** — Point at any folder. Files are plain `.txt`, `.md`, or `.org`. No database, no lock-in.
- **Native Performance** — macOS-native Swift/SwiftUI, optimized for Apple Silicon. Universal binary (arm64 + x86_64).

### Wiki Links (new in 1.0)

- **Wiki-style linking** — `[[target]]` and `[[target|label]]` links between notes, rendered in both editor and preview.
- **Wiki navigation** — Cmd-click in editor or click in preview to open linked notes. Missing notes prompt creation. Ambiguous targets show a picker.
- **Tab-based autocomplete** — Inside `[[...]]`, Tab completes unique matches; ambiguous matches open a suggestion dropdown.

### Editor & Preview

- **Markdown & Org-mode preview** — Toggle with Cmd+P. Scroll position preserved.
- **Markdown table rendering** — Tables render properly in preview.
- **Org-mode support** — `#+TITLE:` displayed as H1, metadata hidden, clean rendering.
- **Font family picker** — Choose editor font from installed monospace fonts.
- **Date insertion** — Cmd+Shift+D inserts today's date (ISO 8601).
- **No auto-substitutions** — Disabled `---` → em-dash and similar unwanted replacements.

### Organization & Navigation

- **Tagging** — Cmd+Shift+T to add inline tags to Markdown and Org files.
- **Mouse back/forward navigation** — Navigate note history with mouse buttons.
- **Per-note cursor memory** — Cursor position preserved when switching notes.
- **Subdirectory creation** — Create files with paths like `projects/2026/notes`; directories auto-created.
- **Right-click rename** — Rename notes from file list context menu.
- **Dual layout** — Vertical and horizontal views (Cmd+Shift+J).

### Desktop Integration

- **Window position memory** — Remembers size and position across launches.
- **Metadata caching** — Instant warm startup (10-30ms).
- **Safe deletes** — Notes move to Trash, not permanently deleted.
- **Read-only indicator** — Visual state for non-writable files.
- **Accessibility labels** — VoiceOver-ready toolbar.
- **Homebrew install** — `brew install --cask msnodderly/tap/neonv`

### Installation

Download the DMG and drag NeoNV to Applications, or:

```bash
brew install --cask msnodderly/tap/neonv
```

**First launch (unsigned app):**
1. Double-click NeoNV — macOS blocks with "cannot be opened"
2. Open **System Settings → Privacy & Security**
3. Click **"Open Anyway"** in Security section
4. Enter password when prompted
