## NeoNV v0.13.0

Fresh app icon with the Neon V2 design.

### New Features

- **New app icon** — Updated to the Neon V2 icon with a refined segmented-tube "N" featuring cyan and magenta neon glow on a dark gradient background

### Installation

Download the DMG and drag NeoNV to Applications.

**First launch (unsigned app):**
1. Double-click NeoNV — macOS blocks with "cannot be opened"
2. Open **System Settings → Privacy & Security**
3. Click **"Open Anyway"** in Security section
4. Enter password when prompted
